TASK: Build “Admin → Brands & Add-ons” with Core / Finish / Hardware adders, CRUD, and guardrails.

Tech context:

- Pricing: base rate (Handmade/Factory) + brand adders per SFT.
- Special rule: Acrylic finish = +₹200/sft (overrides +₹100 finish adder).
- Do NOT touch Replit auth or vite dev helpers.

--------------------------------
1) SHARED SCHEMA (drizzle sqlite)
--------------------------------
In /shared/schema.ts add a brands table:

export const brandTypes = ["core","finish","hardware"] as const;

export const brands = sqlite table "brands":
- id: text pk (nanoid)
- type: text not null (must be one of brandTypes)
- name: text not null, unique within type
- adderPerSft: integer not null default 0   // in ₹/sft
- isDefault: integer not null default 0      // only one default per type
- isActive: integer not null default 1
- createdAt: integer default now
- updatedAt: integer default now

Export types:
- BrandRow (select), NewBrandRow (insert)

Constraint:
- Only one row per type with isDefault=1 (enforce in server logic).

--------------------------------
2) SERVER: Admin Brands API
--------------------------------
Create /server/routes.admin.brands.ts and mount under /api/admin/brands in /server/routes.ts

Endpoints:
- GET  /api/admin/brands           → list (filters: ?type=&active=1&q=)
- POST /api/admin/brands           → create {type,name,adderPerSft,isDefault?,isActive?}
- PUT  /api/admin/brands/:id       → update fields above
- PATCH/PUT /api/admin/brands/:id/default → set this row as default for its type (unset previous)
- PATCH /api/admin/brands/:id/active → {isActive:boolean}
- DELETE /api/admin/brands/:id     → soft delete (isActive=0)

Validation:
- type ∈ ["core","finish","hardware"]
- name 2..60 chars; unique per type (case-insensitive)
- adderPerSft ≥ 0 (integer)
- On setting isDefault=1, ensure all other rows of same type set to 0.

Seeder (optional but helpful) in /server/seed/brands.seed.ts if table empty:
- Core: ["Generic Ply"(default,+0), "Century Ply"(+100), "Greenply"(+100)]
- Finish: ["Generic Laminate (Nimmi)"(default,+0), "Merino"(+100), "Greenlam"(+100), "Acrylic"(+200 special)]
- Hardware: ["Generic"(default,+0), "Hettich"(+100), "Häfele"(+100), "Ebco"(+100), "Sleek"(+100)]

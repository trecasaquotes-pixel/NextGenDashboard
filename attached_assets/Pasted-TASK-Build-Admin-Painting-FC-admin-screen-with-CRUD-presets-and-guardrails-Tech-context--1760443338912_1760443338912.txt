TASK: Build “Admin → Painting & FC” admin screen with CRUD, presets, and guardrails.

Tech context:
- Monorepo: client (Vite React TS), server (Express TS + SQLite via drizzle), shared types in /shared.
- Painting packs are LSUM packages tied to BHK scaling.
- FC “Others” lines are a mix of LSUM (paint) and COUNT (lights, fan hooks, cove).
- Do NOT touch Replit auth or vite dev helpers.

--------------------------------
1) SHARED SCHEMA (drizzle sqlite)
--------------------------------
In /shared/schema.ts add two groups:

A) Painting Packs:
export const paintingPacks = sqlite table "painting_packs" {
  id: text pk (nanoid)
  name: text not null              // e.g., "Tractor Emulsion", "Premium", "Royal Luxury", "Royal Shine", "Royal Aspira"
  basePriceLsum: integer not null  // default LSUM for 3BHK baseline (₹)
  bulletsJson: text not null       // JSON string array of bullet points
  bhkFactorBase: integer not null default 3 // baseline BHK for basePrice (3)
  perBedroomDelta: real not null default 0.10 // +/– 10% per bedroom difference from baseline
  showInQuote: integer not null default 1
  isActive: integer not null default 1
  createdAt: integer default now
  updatedAt: integer default now
}

Types: PaintingPackRow, NewPaintingPackRow.

B) FC Others (admin-configurable):
export const fcCatalog = sqlite table "fc_catalog" {
  id: text pk (nanoid)
  key: text not null unique        // "fc_paint", "fc_lights", "fc_fan_hook", "fc_cove_led"
  displayName: text not null
  unit: text not null              // "LSUM" | "COUNT"
  defaultValue: integer not null default 0 // used to prefill count or lsum
  ratePerUnit: integer not null default 0  // only used when unit=COUNT
  isActive: integer not null default 1
  createdAt: integer default now
  updatedAt: integer default now
}

Types: FCCatalogRow, NewFCCatalogRow.

2) SERVER: ROUTES
--------------------------------
Create /server/routes.admin.paintingFc.ts and mount under /api/admin/painting-fc in /server/routes.ts.

Painting Packs:
- GET    /api/admin/painting-packs         → list (?active=1&showInQuote=1&q=)

FC Catalog:

- PATCH  /api/admin/fc-catalog/:id/active  → {isActive:boolean}
- DELETE /api/admin/fc-catalog/:id         → soft delete

Validation:
- Painting: name 2..60; basePriceLsum ≥ 0; bulletsJson must be array; perBedroomDelta between 0 and 0.25
- FC: key unique, unit in ["LSUM","COUNT"]; for COUNT require ratePerUnit ≥ 0
